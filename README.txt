== Description ==

This plugin for making mobile app for WooCommerce store. You will get WooCommerce, WordPress & Some others API Data using this plugin.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `itech-woo-app.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
